# 使用手册（每日操作SOP）

> 不需要记规则，只需要照着做。每个检查点对着清单执行。

---

## 📅 每日三个强制检查点

| 时间 | 名称 | 检查内容 |
|------|------|---------|
| **BJ 11:45** | 深夜前建仓窗口 | 是否满足建仓条件？ |
| **BJ 17:30** | 死区剪仓评估 | 现有仓位是否需要调整？ |
| **BJ 21:00** | 晨间补仓窗口 | 是否有补仓机会？ |

**这三个时间点是强制的，不是可选的。**

---

## ✅ BJ 11:45 建仓检查清单

进场前确认以下所有问题：

**Step 1：时机确认**
- [ ] 距到期是否在 2–2.5 天以内？（超过2.5天 → 等待）
- [ ] 当前是否不在活跃高峰（BJ 13-16）？（活跃高峰 → 等价格回落）

**Step 2：µ评估**
- [ ] 当前µ是多少？（运行 `/musk-check` 获取）
- [ ] 今日已过会话是否有缺席？（缺席 → 修正µ）
- [ ] 修正后µ对应的中心区间是哪个？

**Step 3：价值比比较**
- [ ] 中心区间当前价格是多少？
- [ ] 中心区间价值比 = 模型概率 / 市场价格 ≥ 1.0？
- [ ] 如果中心区间价格 > 35¢ → 检查两侧区间价值比
- [ ] 比较中心、下方1档、上方1档的价值比，选择最高的那个

**Step 4：退出逻辑**
- [ ] 是到期兑现 or 目标价格止盈？（**必须明确，才能入场**）
- [ ] 止盈目标价是多少？（目标价格止盈模式下）

**Step 5：仓位结构**
- [ ] 主仓：价值比最高区间，50–70% 资金
- [ ] 保护仓：主仓下方1档，20–30% 资金
  - 例外：µ距中心边界 >15条 且 节奏平稳 → 可减少保护仓
- [ ] 高赔率仓（可选）：价值比 ≥ 2.5，≤ 10% 资金，价格 ≤ 5¢

→ **全部通过 → 建仓**
→ **有任何一项不通过 → 等到BJ 21:00再评估**

---

## ⚠️ BJ 17:30 死区评估清单

**每天必做，不能跳过：**

**Step 1：会话检查**
- [ ] 今天下午会话（BJ 0-5）发推数是否正常？（正常 ~14条）
- [ ] 今天傍晚会话（BJ 6-10）是否发生？
- [ ] 深夜会话（BJ 11-16）是否结束？发推数是多少？
- 缺席 → 对应下修µ（见规律库）

**Step 2：µ偏移判断**
- [ ] 当前µ预测 vs 入场时µ预测，偏移是否 > 1.5σ（约25-30条）？
- [ ] 连续2天发推量是否低于均值30%+？（是 → 必须重算µ）

**Step 3：仓位评估**
- [ ] 主仓区间：当前模型是否仍支持这个区间？（µ支持 → 持有）
- [ ] 保护仓区间：是否有必要调整？
- [ ] 有无浮亏 > 20% 且µ已偏出的区间？（有 → 止损）
- [ ] 有无概率 < 15% 的持仓区间？（有 → 强制评估是否出场）

**Step 4：执行**
- 需要调仓 → **立即执行，不推迟**
- 不需要调仓 → 记录当前状态，等BJ 21:00再看

> ⚠️ "再等等看" = 死亡陷阱。死区过后情绪会让你更难出场。

---

## 🌅 BJ 21:00 晨间窗口检查

**Step 1：当前状态**
- [ ] 当前µ是多少？距到期还有多少时间？
- [ ] 主仓是否已建？需要补仓吗？

**Step 2：补仓条件**
- [ ] 有无价值比 ≥ 1.0 的区间未覆盖？
- [ ] 主仓占比是否已到50-70%？（未到 → 可以补仓）

**Step 3：执行**
- 满足条件 → 按仓位结构补仓
- 不满足 → 等到次日BJ 11:45

---

## 🏁 到期前最后操作

**到期前 6–12 小时**：
- [ ] 中心区间价格是否已达 75¢ 以上？
  - 是 → 卖出50%锁定利润，剩余50%博到期$1
  - 否 → 继续持有

**到期前 1–2 小时**：
- [ ] µ已高度确定，如果中心区间 < 75¢ 且模型概率 > 85% → 仍可加仓

---

## 📝 每期结束后（复盘）

1. 打开 [复盘记录_reviews.md](复盘记录_reviews.md)
2. 按7个复盘问题格式填写本期复盘
3. 如有新规律，运行 `/musk-evolve` 触发规则进化检查
4. 更新执行力追踪表

**规则：不复盘就不算完成这一期。**
