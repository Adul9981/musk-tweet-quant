# 入场与仓位框架

> 完整的操作框架：什么时候入场、买什么、买多少、什么时候出场。

---

## 一、µ预测规则

### 1.1 使用最佳µ

🟡 始终使用**最佳µ**（时间加权µ 和 会话修正µ 的均值），而非 C + R×T

简单µ的误差来源：
- 忽略了剩余时间内各小时的发推权重不同
- 没有考虑今日会话是否缺席

### 1.2 µ的不确定性

| 剩余时间 | µ的不确定性（约1σ） | 含义 |
|---------|-----------------|------|
| 3天 | ±50条 | 不可靠，**不要入场** |
| 2天 | ±35条 | 开始有参考价值 |
| 1天 | ±20条 | 较可靠，**适合入场** |
| 12小时 | ±12条 | 高可信，可加仓 |
| 6小时 | ±8条 | 非常可信 |

### 1.3 会话缺席的µ修正

| 缺席会话 | 北京时间 | µ下修 |
|---------|---------|------|
| 深夜会话缺席 | 11点–17点 | **-14条** |
| 下午会话缺席 | 0点–6点   | **-14条** |
| 上午会话缺席 | 20点–24点 | **-11条** |
| 傍晚会话缺席 | 6点–11点  | **-11条** |

🟡 每天BJ 17:30死区评估时，必须检查已完成的会话是否正常。

## 二、入场规则

### 2.1 最佳入场时间窗口

🔴 **铁律：到期前 2–2.5 天入场，不早不晚**

| 入场时机 | 评级 | 原因 |
|---------|------|------|
| BJ 12:00–12:30（深夜爆发前） | 🥇 最强 | 市场未定价深夜爆发，价格洼地 |
| BJ 21:00–21:30（晨间窗口） | 🥈 次强 | 美国上午开始，活跃度回升 |
| 死区之后（BJ 19:00-20:00） | 🥉 可用 | µ清晰，情绪稳定 |
| 活跃高峰中（BJ 13-16） | ⚠️ 慎用 | 价格已被推高，价值比下降 |
| 到期前 < 1天 | ❌ 风险高 | 波动剧烈，除非非常确定 |

### 2.2 入场条件检查清单（三条都满足才入场）

- [ ] 距到期 **2–2.5 天**以内
- [ ] 目标区间价值比 **≥ 1.0**（最好 ≥ 1.2）
- [ ] 当前时机不是活跃高峰（避免追高）
- [ ] 对比中心区间和两侧相邻区间的价值比，选择价值比**最高**的区间入场
  - 如果中心区间价格 > 35¢，必须显式比较两侧是否更划算

### 2.4 精度与价格的矛盾解法

**核心矛盾**：越早入场价格越低，但µ越不准；越晚入场µ越准，但中心价格可能已被推高。

```
越早入场 → 价格低  ↔  µ不确定性大（可能买错档）
越晚入场 → 准确度高 ↔  中心区间可能已 > 35¢
```

**µ不确定性对照表（精度参考）**：

| 剩余时间 | µ不确定性（±1σ） | 档位误差 |
|---------|--------------|--------|
| 3天 | ±50条 | ±2.5档，不知哪档是真正中心 |
| 2天 | ±35条 | ±1.75档，有参考价值但不稳 |
| 1天 | ±20条 | ±1档，较可靠 |
| 12小时 | ±12条 | ±0.6档，高可信 |
| 6小时 | ±8条 | ±0.4档，非常可信 |

**三种解法**：

**解法1 — 在2-2.5天窗口内，选 BJ 11:45 而非任意时间入场**
- BJ 12:00 前市场按当前低速率（1.2条/h）定价，未计入即将到来的深夜爆发（3.0条/h）
- BJ 11:45 买入的中心价格，比 BJ 14:00 高峰时低 10–20¢
- "2天时中心价格很贵"大多是在 BJ 13–16 高峰期追入导致的

**解法2 — VR 是真正的入场开关，中心贵就转买相邻区间**
- 中心价格高 ≠ 不能入场；VR < 1.0 才等于不该入场
- 模型有 +0.3档系统性偏高 → 下方1档 VR 经常优于中心
- 典型案例（5月27日）：中心 44¢ VR=0.73 ❌，220-239 区间 6.5¢ VR=1.54 ✅
- 操作：主仓移到 VR 最高的相邻区间，不必强买中心

**解法3 — 跨期分层（既要低价，又要精度）**
- 到期 3–4天：**轻仓10-20%** 提前布局，抢低价，接受较大不确定性
- 到期 1.5–2天：**主力仓位55-70%** 在µ可靠后集中建仓
- 两层配合：用轻仓的低成本优势 + 主仓的高精度优势，而非二选一

🟡 **结论：2-2.5天规则控制"µ什么时候够准"，VR 规则控制"这个价格值不值买"，二者缺一不可。矛盾的本质不是时机问题，是价格合理性问题。**

### 2.3 边界分仓规则

🟡 若最佳µ距某区间边界 **≤ 10条**：
- 各50%分在两侧区间
- 理由：µ本身有±10条的预测误差，边界两侧都有实质概率

🟡 **µ不确定时两侧覆盖**：µ不确定性 > 20条时（剩余时间 > 1.5天），中心落点两侧都要有仓位，不能只押一侧。（来源：5月26日复盘）

## 三、仓位管理规则

### 3.1 三层仓位结构

🔴 **铁律：仓位集中，不分散买5个区间**

| 仓位 | 目标 | $2,500账户参考金额（括号内为比例） |
|------|------|--------------------------------|
| 🎯 **中心落点仓** | 鱼腹收益：~30¢涨到~70¢，稳定正EV | $130–180（约55%） |
| 🛡️ **保护仓** | 确保这期不会亏损严重 | $90–120（约37%） |
| 🎰 **彩票仓** | 完全可以亏光，博巨额惊喜 | $20–40（≤8%） |
| **单期合计上限** | — | **$250–340（约10–13%账户）** |

**中心落点仓选法**：µ中心区间，VR ≥ 1.0，价格 ≤ 35¢；中心价格 > 35¢ 时转买下方1档（VR通常更高）

**保护仓选法**：中心落点**下方1档**——模型有+0.3档系统性偏高，下方风险更大；33期回测覆盖中心+下方1档命中率约65%+

**彩票仓选法**：YES ≤ 2¢，区间下沿在 µ ~ µ+60条范围内，还没被触达

> ⚠️ **仓位管理铁律**：每期开始前先划定本期总额，账户里多余的钱提现到链上钱包。想加仓必须主动转账，摩擦成本本身就是最好的纪律。

### 3.2 分层建仓节奏

| 阶段 | 距到期 | 操作 | 金额参考（$2,500账户） |
|------|--------|------|----------------------|
| 第一层 | 第3天开始 | 轻仓布局，价格低但µ未稳 | $80–130（单期上限的30–40%） |
| 第二层 | 第1–2天 | 主力跟进，µ可信度大幅提升 | $170–210（剩余60–70%） |
| 彩票仓 | 任意时机 | 满足条件即入，独立于主仓节奏 | $20–40（固定上限） |

### 3.3 调仓规则

🟡 **µ偏移超过 1.5σ（约25-30条）时才考虑调仓**，不要每天重算换仓。

调仓时机只有两个：
1. **死区（BJ 17:30–19:00）**：最理性，µ稳定，不受活跃期情绪干扰
2. **确认µ大幅偏移**（超过1.5σ），不是"感觉可能要变"

🔴 **连续降温触发规则**：Musk连续2天发推量低于历史均值30%+，视为µ下修信号，必须在当天死区内重算µ并评估调仓。不能等"再看一天"。

## 四、出场规则

### 4.1 止盈窗口

🟡 **BJ 14:00–15:00（深夜高峰峰值）是最佳止盈评估时间**

- 全天推文速率最高，µ此时最接近真实终点
- 市场此时价格最高（中心区间可能已到70–80¢）
- 如果已达目标收益，可考虑部分止盈

### 4.2 止损与剪仓

🔴 **铁律：死区（BJ 17:30–19:00）是唯一强制执行剪仓的窗口**

死区剪仓原则：
- 对概率 < 15% 的区间 → 强制评估，大概率应该出
- 浮亏 > 20% 且µ已偏出区间 → 接受损失，止损出场
- 「再等等看」= 死亡陷阱，死区过后下一个活跃期会让你更难出场

### 4.3 到期前最后操作

🔵 到期前 6–12 小时，如果中心区间价格已达 75¢ 以上：
- 可以选择锁定部分利润（卖出50%）
- 剩余50%博到期全额（$1）

### 4.4 入场前必须设好退出逻辑

🟡 每笔仓位入场前必须明确：**到期兑现 or 目标价格止盈**。没有退出逻辑就不入场。

## 五、执行纪律

### 5.1 不追节奏

🔴 **铁律：马斯克当日发推异常，不临时改变主仓位**

- 异常多：市场追高，追买等于追高
- 异常少：单日异常不改变整期µ

唯一需要调整的信号：µ连续2天偏离超过1.5σ。

### 5.2 不依赖主观判断

🟡 所有操作必须提前设定条件，对照条件执行：

```
✅ 对：「如果价值比≥1.2且剩余时间在2天内，执行建仓」
❌ 错：「感觉今天节奏不对，再等等看」
```
