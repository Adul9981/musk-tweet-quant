# 入场与仓位框架

> 完整的操作框架：什么时候入场、买什么、买多少、什么时候出场。

---

## 一、µ预测规则

### 1.1 使用最佳µ

🟡 始终使用**最佳µ**（时间加权µ 和 会话修正µ 的均值），而非 C + R×T

简单µ的误差来源：
- 忽略了剩余时间内各小时的发推权重不同
- 没有考虑今日会话是否缺席

### 1.2 µ的不确定性

| 剩余时间 | µ的不确定性（约1σ） | 含义 |
|---------|-----------------|------|
| 3天 | ±50条 | 不可靠，**不要入场** |
| 2天 | ±35条 | 开始有参考价值 |
| 1天 | ±20条 | 较可靠，**适合入场** |
| 12小时 | ±12条 | 高可信，可加仓 |
| 6小时 | ±8条 | 非常可信 |

### 1.3 会话缺席的µ修正

| 会话缺席 | µ下修 |
|---------|------|
| 深夜会话缺席（BJ 13-16无推文） | **-14条** |
| 下午会话缺席（BJ 0-5异常低）   | **-14条** |
| 上午会话缺席                  | **-11条** |
| 傍晚会话缺席                  | **-11条** |

🟡 每天BJ 17:30死区评估时，必须检查已完成的会话是否正常。

## 二、入场规则

### 2.1 最佳入场时间窗口

🔴 **铁律：到期前 2–2.5 天入场，不早不晚**

| 入场时机 | 评级 | 原因 |
|---------|------|------|
| BJ 12:00–12:30（深夜爆发前） | 🥇 最强 | 市场未定价深夜爆发，价格洼地 |
| BJ 21:00–21:30（晨间窗口） | 🥈 次强 | 美国上午开始，活跃度回升 |
| 死区之后（BJ 19:00-20:00） | 🥉 可用 | µ清晰，情绪稳定 |
| 活跃高峰中（BJ 13-16） | ⚠️ 慎用 | 价格已被推高，价值比下降 |
| 到期前 < 1天 | ❌ 风险高 | 波动剧烈，除非非常确定 |

### 2.2 入场条件检查清单（三条都满足才入场）

- [ ] 距到期 **2–2.5 天**以内
- [ ] 目标区间价值比 **≥ 1.0**（最好 ≥ 1.2）
- [ ] 当前时机不是活跃高峰（避免追高）
- [ ] 对比中心区间和两侧相邻区间的价值比，选择价值比**最高**的区间入场
  - 如果中心区间价格 > 35¢，必须显式比较两侧是否更划算

### 2.3 边界分仓规则

🟡 若最佳µ距某区间边界 **≤ 10条**：
- 各50%分在两侧区间
- 理由：µ本身有±10条的预测误差，边界两侧都有实质概率

🟡 **µ不确定时两侧覆盖**：µ不确定性 > 20条时（剩余时间 > 1.5天），中心落点两侧都要有仓位，不能只押一侧。（来源：5月26日复盘）

## 三、仓位管理规则

### 3.1 三层仓位结构

🔴 **铁律：主仓集中，不分散买5个区间**

| 仓位 | 比例 | 区间 | 条件 |
|------|-----|------|------|
| **主仓** | 50–70% | 价值比最高的区间（通常是中心落点或其相邻区间） | 价值比 ≥ 1.0 |
| **保护仓** | 20–30% | 中心落点**下方1档**邻区 | 模型有偏高倾向（+0.3档系统性偏差），下侧风险更大 |
| **高赔率仓** | ≤10% | 价值比 ≥ 2.5 的低价区间 | 可选，有合适目标时加 |

**为什么要加保护仓**：
- 33期回测：模型存在 +0.3档的系统性偏高
- 只押中心1档命中率38%；覆盖中心+下方1档覆盖率约65%+
- 成本：平摊资金，单档收益降低，但整体EV显著提升

🟡 **何时不需要保护仓**：µ与中心区间边界相差 > 15条，且当期节奏平稳（偏差 < ±15%）。

### 3.2 高赔率补充仓规则

满足以下**全部条件**时，用小仓位（≤10%利润）押高赔率区间：
- 盈亏比 ≥ 20:1（买入价 ≤ 5¢）
- **你相信真实概率 > 盈亏平衡概率**（不能只看赔率高就买）
- 模型概率与市场定价的差距在合理范围（价值比 0.5x 以上）
- 已有主仓在盈利中（用利润博，不用本金博）

### 3.3 调仓规则

🟡 **µ偏移超过 1.5σ（约25-30条）时才考虑调仓**，不要每天重算换仓。

调仓时机只有两个：
1. **死区（BJ 17:30–19:00）**：最理性，µ稳定，不受活跃期情绪干扰
2. **确认µ大幅偏移**（超过1.5σ），不是"感觉可能要变"

🔴 **连续降温触发规则**：Musk连续2天发推量低于历史均值30%+，视为µ下修信号，必须在当天死区内重算µ并评估调仓。不能等"再看一天"。

## 四、出场规则

### 4.1 止盈窗口

🟡 **BJ 14:00–15:00（深夜高峰峰值）是最佳止盈评估时间**

- 全天推文速率最高，µ此时最接近真实终点
- 市场此时价格最高（中心区间可能已到70–80¢）
- 如果已达目标收益，可考虑部分止盈

### 4.2 止损与剪仓

🔴 **铁律：死区（BJ 17:30–19:00）是唯一强制执行剪仓的窗口**

死区剪仓原则：
- 对概率 < 15% 的区间 → 强制评估，大概率应该出
- 浮亏 > 20% 且µ已偏出区间 → 接受损失，止损出场
- 「再等等看」= 死亡陷阱，死区过后下一个活跃期会让你更难出场

### 4.3 到期前最后操作

🔵 到期前 6–12 小时，如果中心区间价格已达 75¢ 以上：
- 可以选择锁定部分利润（卖出50%）
- 剩余50%博到期全额（$1）

### 4.4 入场前必须设好退出逻辑

🟡 每笔仓位入场前必须明确：**到期兑现 or 目标价格止盈**。没有退出逻辑就不入场。

## 五、执行纪律

### 5.1 不追节奏

🔴 **铁律：马斯克当日发推异常，不临时改变主仓位**

- 异常多：市场追高，追买等于追高
- 异常少：单日异常不改变整期µ

唯一需要调整的信号：µ连续2天偏离超过1.5σ。

### 5.2 不依赖主观判断

🟡 所有操作必须提前设定条件，对照条件执行：

```
✅ 对：「如果价值比≥1.2且剩余时间在2天内，执行建仓」
❌ 错：「感觉今天节奏不对，再等等看」
```
