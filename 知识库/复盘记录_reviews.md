### 期次：5月22日–5月29日（2026年5月29日到期）· 复盘

**最终结果**：落点区间 **180–199**，持仓 **160–179**，盈利（具体金额未记录）

**核心事件时间线**

| 时间节点 | 市场状态 | 操作 |
|---------|---------|------|
| 到期前2天12小时 | 180–199 飙升至 40–50¢，盈亏比极差 | ✅ **正确：未买入中心** |
| 同一时间 | 160–179 价格仅 13¢，VR 明显更高 | ✅ **正确：识别并买入** |
| 最后1天 | 160–179 飙升至 70–80¢ | ❌ **失误：未止盈** |
| 最后5小时 | 发推数在 174 停顿较长，µ仍指向 160–179 | — 持有 |
| 最后3小时 | Musk 连发，发推数从 174 跳至 180+ | 落点跨越区间边界 |
| 到期前 | 180–199 从 20¢ 飙升至 100¢（5倍） | ❌ **仓位太小，未获超额收益** |

**问题拆解（7个复盘问题）**

| 问题 | 回答 |
|------|------|
| 判断对了什么？ | ① 正确识别中心区间（180–199）被高估，VR<1.0，未追入；② 主动找到价值比更高的 160–179（13¢），逻辑清晰 |
| 最懊悔的操作？ | 160–179 飙至 70–80¢ 时没有止盈；高赔率机会（180–199 在 20¢ 时）仓位太小 |
| 当时为什么没执行正确操作？ | 70–80¢ 时贪心继续持有博$1；20¢ 买入 180–199 时因主仓在 160–179，心理上没有余量加注 |
| 如果重来，哪个时间点做什么不同？ | ① 160–179 涨至 70¢+ 时减仓 50% 锁利；② 最后5小时发推在 174 停顿时，识别边界风险，用小仓位（≤10%）同时覆盖 180–199 |
| 最终盈亏 | 盈利（有收益但未最大化），主仓在 160–179 最终落空，无缘 5 倍区间 |
| 印象深刻的市场信号？ | 180–199 在 20¢ 时整个市场低估了最后3小时的爆发风险；发推数在 174 停顿是边界风险信号，当时未警觉 |
| 执行力评分 | **6/10** — 入场判断优秀，中盘正确，结尾两个操作失误（未止盈+边界风险未对冲） |

**本期最有价值的思考（需沉淀进知识库）**

> **核心洞察：VR 驱动选仓的正确性**
> 中心区间 40–50¢ 是陷阱，VR<1.0；邻档 13¢ 是机会，VR 明显更高。
> 这个判断完全正确，且在最后关头被市场验证——只是错在没有覆盖边界风险。

> **边界风险处理规则（新增）**
> 发推数在区间边界附近停顿（距上沿 ≤ 10条）时，需用小仓位（≤10%）同时覆盖相邻区间。
> 本期 174 停顿时，距 180 仅 6 条，这是明确的边界信号，应触发边界分仓规则。

**从本期提炼的规则**

| 规则等级 | 内容 |
|---------|------|
| 🟡 止盈纪律 | 持仓区间价格达 70¢+ 且距到期 < 12小时，应减仓 50% 锁利，不博满额 |
| 🟡 边界停顿预警 | 发推数在区间上沿附近停顿（距边界 ≤ 10条，停顿 ≥ 30分钟），必须检查是否需要覆盖相邻区间 |
| 🟡 高赔率仓留余量 | 主仓建立后，保留 ≤10% 资金专门用于高赔率机会，而不是全部压在主仓 |

### ⭐ 期次：5月19日–5月26日（2026年5月26日到期）· 完整复盘

**最终结果**：落点区间 240–259，亏损约 **$1000+**

**完整错误链条（按时间顺序）**

**① 根因：入场早了整整一天**
- 在到期前 **3天8小时** 就大量买入 300+ 区间，违反2–2.5天铁律
- 后果：多承受了整整一个「Musk突然变冷」的风险窗口
- 如果2天前入场，Musk降温信号已经发生，根本不会买那些高区间

**② µ下修信号来了，没有执行调仓**
- Musk在第3天和第2天突然降温，这是明确的µ下修信号
  - 深夜会话缺席 → µ下移14条
  - 连续两天偏低 → 偏移超过1.5σ
- 没有量化确认，陷入「再等等看」，错过多个死区调仓窗口

**③ 260–279 没有对冲保护仓**
- 买了260–279 和 280+ 区间，但完全没有在 240–259 建保护仓
- µ不确定性 > 20条时，中心落点两侧都应该有覆盖
- 一旦Musk降温就完全没有对冲，全靠一边

**④ 在亏损区间不断加仓（沉没成本谬误）**
- 280+ 区间已明显偏离µ，但继续加仓
- 心理动因：「已经亏了这么多，万一涨回来呢？」
- 正确逻辑：加仓的唯一依据是「当前µ是否支持这个区间」，浮亏多少与此完全无关

**⑤ 1天18小时时的调仓窗口识别到了但没执行**
- 看到了机会：260–279 → 250–269，但没动
- 原因：差距已拉开，心理成本太高，拖延效应发作
- 正确做法：跑 `/musk-check` 让数字告诉你，不要靠主观感受决策

**⑥ 3¢抄底方向对，但仓位太小且卖出过早**
- 做对的事：240–259 跌到 3¢ 时识别到价值，买入4000份（约$120）
- 仓位太小：当时确信µ落这里，但只花了$120
- 卖出过早：在 20–40¢ 卖出了1000+份——入场前没有明确退出逻辑

**问题拆解（7个复盘问题）**

| 问题 | 回答 |
|------|------|
| 判断对了什么？ | 最后识别到了240–259是落点，3¢时买入方向正确 |
| 最懊悔的操作？ | 在3天8小时时就重仓买入300+高区间，是本期所有问题的根源 |
| 当时为什么没执行正确操作？ | 进场早 → Musk降温 → 不肯认错 → 沉没成本加仓 → 越陷越深 |
| 如果重来，哪个时间点做什么不同？ | ① 2天前才入场；② Musk连续降温后的死区立刻重算µ调仓；③ 入场时在低区间同步建保护仓 |
| 最终盈亏 | 亏损约 $1000+，远超预期 |
| 印象深刻的市场信号？ | 240–259 在最后一天跌至3¢，是极强的价值信号——模型早就显示µ在这里 |
| 执行力评分 | **2/10** — 判断力没有完全失效，但规则违反5条 |

**根因总结**：
> 本期所有失误的根源是**入场早了一天**，引爆了后续的µ风险暴露。

### 期次：5月12日到期（2026年5月）

**结果**：100–119 区间最终涨至 85¢（入场价仅 5¢，17倍收益被错过）

**核心失误**：价值比 = 3.0，模型概率显著，但没有买入

**根因分析**：
- 对小概率区间的心理排斥——虽然价值比高，但"感觉不太可能"
- 没有执行「高赔率补充仓」规则（≤10%小仓位押注）

**教训**：
> 价值比 ≥ 2.0 且概率不低时，必须用小仓位押注，不管直觉如何。这是规则，不是感觉问题。

**执行力评分**：4/10

## 🧠 从复盘提炼的核心认知

### 关于执行
1. **判断正确但不执行 = 判断错误**，结果一样亏钱
2. **差距拉大后更难执行**，这是人性，解决方案是在差距小时就执行
3. **死区（BJ 17:30）是最理性的执行窗口**，每天只有这一次

### 关于建仓
4. **到期前2–2.5天入场**，不早不晚
5. **主仓只买一个区间**，集中比分散好
6. **价值比≥2.0的高赔率区间必须小仓押注**，不管直觉感觉

### 关于止盈
7. **全天峰值是 BJ 14:00–15:00（深夜高峰）**，这是最佳止盈评估窗口
8. 不要在活跃期（BJ 13–16）做减仓决定，情绪容易被波动带走
