## ⚡ 期末 NO 埋伏策略（2026-06-02 实战验证）

> 这是目前验证过的、可重复执行的高胜率策略。

### 触发条件（全部满足才入场）

| 条件 | 标准 |
|------|------|
| 时机 | 到期当天，BJ 11:30–12:00 之前入场 |
| 推文进度 | 推文数在某区间内，距上沿 ≥ 20条（不是快突破，是还有空间） |
| 价格 | NO 价格 ≤ 15¢（市场极度低估突破概率） |
| 时段 | 入场后至少覆盖一个活跃窗口（BJ 12–16 或 BJ 00–04） |
| 速率 | 当日发推速率 ≥ 历史均值（今天不是低谷日） |

### 操作结构

```
入场：买目标区间 NO，≤ 20% 仓位
出本：价格达到 +100% 时，卖出本金部分
持利：利润仓零成本持有，等以下出场信号之一：
  - 推文速率连续 30 分钟归零
  - BJ 16:00 活跃窗口关闭
  - NO 价格 > 80¢（上涨空间不足）
  - 到期自然兑现
```

### 为什么长期正EV

1. **市场定价偏差**：多数人押「会落哪个区间」，导致 NO 被系统性低估
2. **时间优势**：活跃窗口可预测，提前埋伏而非事后追
3. **出本留利结构**：本金锁定后，利润仓是零成本期权，持有不亏

| 日期 | 入场 | 入场价 | 出本时价格 | 最终结果 |
|------|------|--------|-----------|------|
| 2026-06-02 AM | 5月26–6月2期 NO，推文142条 | 12¢ | 26¢（+126%） | 利润仓到期兑现，合计+~$200 |
| 2026-06-02 PM | 140–159 NO，期末2h爆发 | ~11¢ | 翻倍出本 | NO到期兑现100¢，总收益+~$300 |

## 💥 多区间联动策略（2026-06-02 晚 验证）

> 当马斯克在期末集中爆发时，相邻区间会产生连锁价格运动，可以双腿布局。

### 联动逻辑

```
当前区间 [A, B]  推文数接近B上沿
  → 买 [A,B] NO（低价，押突破）
  → 同步买 [B,C] YES（低价，押落点在下一区间）

两腿均能获利：
  NO腿：推文突破B → 趋近100¢
  YES腿：推文落在[B,C] → 升值
```

### 触发信号

| 信号 | 标准 |
|------|------|
| 推文进度 | 距当前区间上沿 ≤ 15条 |
| 速率 | 过去1小时发推速率 ≥ 5条/h |
| 下方YES价格 | 上方区间YES ≤ 15¢（上涨空间大） |
| 时间余量 | 距到期 ≥ 30分钟 |

### 彩票仓：极低价YES

> 价格 ≤ 1¢ 的远端YES区间，分配 ≤ $50，是零成本期权。

- 入场成本极低，最坏归零不影响总仓
- 发生极端爆发时（如+19条/2h），回报可达 30-40倍
- **不是主仓，是附属博弈**

## 🎰 彩票仓策略（2026-06-02 晚 首次验证）

> 0.7¢ → 28¢ = +3900%。这不是运气，是可以识别的系统性低估。

### 核心逻辑

市场对极端区间（远端YES）的定价，基于的是**静态历史分布**，没有反映：
1. 当日节奏是否明显偏活跃（热力图可以看到）
2. 剩余时间内是否覆盖高频时段
3. 当前已接近某区间上沿时连锁突破的概率

**结果：越远的区间，低估越严重。0.7¢ YES通常真实概率远高于0.7%。**

### 触发条件（同时满足）

| 条件 | 标准 | 数据来源 |
|------|------|---------|
| 当日节奏偏高 | 今日已发推数 ≥ 历史日均的 1.2倍 | 热力图/API |
| 目标区间价格 | YES ≤ 1¢（或 NO≥99¢） | 盘口 |
| 距上方区间下沿 | 当前推文数距目标区间下沿 ≤ 50条 | 当前推文数 |
| 时间余量 | 距到期 ≥ 30分钟 | 剩余时间 |
| 高频时段 | 剩余时间覆盖至少一个活跃窗口 | BJ时间 |

### 操作结构

```
仓位：总账户 ≤ 2%（这是彩票，不是主仓）
  推荐绝对金额：$20–$100
  推荐份数：按0.7¢算，$50可以买约7000份

出场选择（三选一）：
  A. 到期自然兑现（最优，适合确信度高的情况）
  B. 价格达到 10–15¢ 时止盈（稳健，约10-20倍）
  C. 出本留利（达到+100%时卖出本金，利润仓持有）
```

### 为什么正EV

- 成本极低，最坏归零也影响不大
- 市场系统性低估极端区间
- 活跃日的节奏信号可以提前识别
- 无需等待，买入后持有到期即可

| 2026-06-02 | 180–199 YES | 0.7¢ | 28¢（回落12¢） | **40x峰值** | 当日已发45条，节奏偏高 |

## 🔴 一句话铁律

1. 到期前 >3天不入场
2. 价值比 <1.0 不入场
3. 死区强制剪仓评估（不推迟）
4. 连续2天降温→死区重算µ
5. 加仓唯一依据：µ支持这个区间（浮亏无关）
6. 主仓集中，不分散买5个区间
7. **买NO时，同步检查上方YES价格** → ≤15¢就配双腿
8. **今日节奏 ≥ 1.2倍历史均值 + 远端YES ≤ 1¢** → 布局彩票仓 ≤$50，不要错过
9. 热力图当天看：今天偏高还是偏低？这直接决定当期落点上修或下修
