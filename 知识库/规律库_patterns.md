## ⏰ 时间规律

| 北京时间 | 含义 | 操作 |
|---------|------|------|
| **BJ 11:45–12:15** | 深夜爆发前，价格最低洼地 | ✅ 建仓 |
| **BJ 13:00–16:00** | 全天最高峰（日均3.0–3.4条/h） | 📊 评估止盈 |
| **BJ 17:30–19:00** | 死区，全天最低（0.8–1.2条/h） | ⚠️ 强制剪仓评估 |
| **BJ 21:00–21:30** | 晨间窗口，美国上午开始 | ✅ 建仓/补仓 |
| **BJ 08:00–12:00** | 美国傍晚，低谷（不是高峰！） | ❌ 不要在此等信号 |

**BJ 12→13 跳跃**：+150%，全天最大单小时跳跃，每天稳定发生（206天数据）

## 🎯 入场规律

| 规律 | 数字 |
|------|------|
| 最佳入场时机 | 到期前 **2–2.5天** |
| 最早入场 | 到期前 **2.5天**（不超过） |
| 不能入场 | 到期前 **>3天**（µ误差±50条） |
| 价值比门槛 | **≥ 1.0**（最好 ≥ 1.2） |
| 中心区间价格警戒线 | **> 35¢**（必须比较两侧） |

## 🔴 一句话铁律

1. 到期前 >3天不入场
2. 价值比 <1.0 不入场
3. 死区强制剪仓评估（不推迟）
4. 连续2天降温→死区重算µ
5. 加仓唯一依据：µ支持这个区间（浮亏无关）
6. 主仓集中，不分散买5个区间
