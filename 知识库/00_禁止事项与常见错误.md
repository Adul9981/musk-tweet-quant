# 禁止事项与常见错误

> 这里只放"付出过代价"的教训。来自真实失误，不是理论。
> 操作前扫一遍，特别是进场前。

---

## 🔴 六条铁律（绝不违反）

| # | 铁律 | 违反后果 |
|---|------|---------|
| 1 | **到期前超过3天不入场** | µ误差±50条，区间判断不可靠，是赌博不是投资 |
| 2 | **主仓集中，不分散买5个区间** | 边缘区间大概率亏损，吃掉中心区间盈利 |
| 3 | **中心落点不一定是最值得买的区间** | 预测正确≠EV正，必须同时满足价值比≥1.0 |
| 4 | **马斯克当日发推异常，不临时改变主仓位** | 单日异常是噪音，µ需连续2天偏离1.5σ才调仓 |
| 5 | **死区（BJ 17:30–19:00）强制评估剪仓，不推迟** | 拖过死区→下一个活跃期情绪影响→永远出不了 |
| 6 | **连续2天降温必须重算µ并在死区评估调仓** | 拖延=沉没成本加深，越陷越深（来自5月26日血泪教训） |

## ❌ 反面教材（绝对不要做的事）

| 错误行为 | 后果 | 正确做法 |
|---------|------|---------|
| 提前3天以上入场 | 噪音极大，被虚晃 | 到期前2–2.5天 |
| 分散买5个区间 | 边缘亏损吃掉中心盈利 | 集中买1-2个区间（主仓+保护仓） |
| 在活跃高峰追买 | 追高，价值比已下降 | BJ 12:00前建仓 |
| 看到差距拉大就拖延调仓 | 越拖越难出 | 死区强制执行 |
| 凭感觉放弃高赔率小仓 | 错过17倍收益（5月12日案例） | 价值比≥2.5必须押 |
| 当日节奏异常就改主仓 | 被短期噪音带跑 | 看µ偏移量，不看单日 |
| 空仓等"完美时机" | 永远等不到 | 价值比≥1.0即可入场 |
| 亏损区间继续加仓 | 把小亏变大亏 | 加仓唯一依据：µ支持这个区间 |
| 入场前没有退出逻辑 | 中途乱卖或死拿 | 入场前先定好：到期兑现 or 目标价止盈 |
| 买了"最可能"区间但价格过高 | 正确预测方向，仍然亏钱 | 入场前必查价值比，中心区间>35¢时先看两侧 |
| 只看方向，不看价格 | 长期来看必然负EV | 价值比 < 1.0 的区间坚决不入场 |
| µ不确定时只押一侧 | Musk降温后完全无对冲 | µ不确定性>20条时两侧都要有仓位 |

## 🧠 认知偏差速查（每次操作前对照）

| 陷阱 | 触发场景 | 规则 |
|------|---------|------|
| **距离幻觉** | "只差X条了，感觉快赢了" | 「近」不是概率。区间边界高溢价=卖出信号，不是持仓信号 |
| **叙事屈服** | 听到反驳后直觉上想改立场 | 任何改变判断的理由，先跑数字再开口，不能先表态 |
| **亏损追仓** | "用赚的钱赌一把弥补亏损" | 盈利仓和亏损仓是两笔独立交易，不能互相抵消 |
| **概率不自洽** | 认为市场错了，但用市场概率算EV | 认为市场错→必须用模型概率算EV，不能混用 |
| **拖延效应** | 差距越大越不想动仓 | 调仓只看µ是否支持，不看浮亏多少。死区是最后窗口 |
| **已有暴露** | "这个区间值得押，我要加仓" | 先问：已有仓位是否已覆盖这个暴露？额外加仓边际价值是什么？ |
| **方向正确幻觉** | "我押对了方向，结果还是亏" | 预测正确≠下注正确。必须同时满足：概率正确+价格合理（价值比≥1.0） |
